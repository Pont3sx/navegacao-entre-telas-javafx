# Navegação Entre Telas - JavaFX

Projeto prático desenvolvido em JavaFX com FXML para demonstrar navegação entre telas utilizando Scene Builder e arquitetura separada entre interface e lógica.

## Tecnologias Utilizadas

- Java 21
- JavaFX
- Maven
- Scene Builder
- IntelliJ IDEA

## Funcionalidades

- Navegação entre telas
- Utilização de FXML
- Estrutura organizada em pacotes

## Estrutura do Projeto

```bash
src/
 └── main/
      ├── java/
      │    └── org/example/
      │         ├── controller/
      │         │       └── LoginController
      │         └── Main.java
      └── resources/
           └── fxml/
                ├── TelaLogin.fxml
                └── TelaPrincipal.fxml
